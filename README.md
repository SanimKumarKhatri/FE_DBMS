# Front End
To run this following packages nees to be installed:
```
yarn add next react react-dom
```
```
npm install @nextui-org/react
```
```
npm i react-iconly
```
```
npm install --save-dev nodemon
```
```
npm install cookie-parser
```
```
npm install axios
```
To run the fron end
```
npm run dev
```
